Règles et base du jeu

=======




Règles du Mahjong

Le solitaire de Mahjong est un jeu de puzzle gratuit basé sur un jeu chinois traditionnel pour quatre personnes. Le but est de se débarrasser de toutes les 144 tuiles de la table. Vous pouvez éliminer des tuiles libres seulement si elles sont appariées. La tuile est libre quand il n'y a aucune tuile de plus vers la gauche ou vers la droite de celle-ci. Les tuiles appariées sont n'importe quelles deux tuiles exactement identiques.

Le plan du jeu est d’éliminer toutes les tuiles de jeu dans le temps le plus rapide, en essayant d'atteindre le plus haut score. Chaque tuile a une image; il y a 42 images diverses en total. Les tuiles doivent être choisies et combinées avec d'autres tuiles de la même image. Chaque fois que vous combinez deux tuiles, elles disparaissent toutes les deux, quand toutes les tuiles ont disparu le jeu est gagné.
